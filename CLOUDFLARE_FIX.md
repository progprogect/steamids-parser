# Решение проблемы Cloudflare

## Проблема
Cloudflare блокирует headless браузеры, возвращая 403 Forbidden.

## Решение

### Вариант 1: Первый запуск с headless=False
1. Установите `HEADLESS = False` в `config.py`
2. Запустите парсер: `python parser.py`
3. Когда откроется браузер, пройдите Cloudflare challenge вручную
4. Cookies будут сохранены автоматически
5. После этого можно вернуть `HEADLESS = True`

### Вариант 2: Использовать stealth режим
Можно попробовать использовать библиотеки для обхода Cloudflare:
- `playwright-stealth`
- Настройки браузера для имитации реального пользователя

### Вариант 3: Использовать прокси/VPN
Если Cloudflare блокирует по IP, можно использовать прокси.

## Текущий статус
Парсер работает, но требует прохождения Cloudflare challenge при первом запуске.
После сохранения cookies последующие запуски должны работать автоматически.

## Проверка работы
После прохождения Cloudflare:
1. Cookies сохранятся в `data/cookies.json`
2. API запросы должны перехватываться через `page.on("response")`
3. Данные должны сохраняться в базу данных
