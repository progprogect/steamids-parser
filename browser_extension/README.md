# SteamDB Parser Browser Extension

Расширение для браузера, которое помогает парсеру обойти Cloudflare, используя вашу активную сессию браузера.

## Установка

### Chrome/Edge:
1. Откройте `chrome://extensions/` (или `edge://extensions/`)
2. Включите "Режим разработчика" (Developer mode)
3. Нажмите "Загрузить распакованное расширение" (Load unpacked)
4. Выберите папку `browser_extension`

### Firefox:
1. Откройте `about:debugging`
2. Нажмите "Этот Firefox" (This Firefox)
3. Нажмите "Загрузить временное дополнение" (Load Temporary Add-on)
4. Выберите файл `manifest.json` из папки `browser_extension`

## Использование

1. Установите расширение
2. Откройте SteamDB в браузере и пройдите Cloudflare challenge (если требуется)
3. Расширение автоматически будет перехватывать API запросы и сохранять данные
4. Или используйте кнопку "Parse Current Page" в popup расширения
5. Экспортируйте данные через кнопку "Export Data"

## Интеграция с парсером

Парсер может использовать данные, сохраненные расширением, или расширение может отправлять данные напрямую в парсер через API.
