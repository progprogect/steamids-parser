# Быстрый старт

## 1. Установка зависимостей

```bash
pip install -r requirements.txt
playwright install chromium
```

## 2. Подготовка APP IDs

Отредактируйте `app_ids.txt` и добавьте APP IDs (по одному на строку):
```
730
440
570
```

## 3. Запуск

```bash
python parser.py
```

## Что происходит:

1. ✅ Создается база данных `data/steam_data.db`
2. ✅ Загружаются APP IDs из `app_ids.txt`
3. ✅ APP IDs разбиваются на батчи по 10
4. ✅ Запускается 10 параллельных потоков
5. ✅ Каждый поток обрабатывает батч через Compare инструмент
6. ✅ Данные сохраняются в БД
7. ✅ Отображается прогресс в реальном времени

## Ожидаемая скорость:

- **225,000 - 450,000 APP IDs/час**
- **12-24 минуты на 100,000 APP IDs**

## Остановка:

Нажмите `Ctrl+C` - парсер корректно завершит работу и сохранит checkpoint.

## Возобновление:

Просто запустите снова - парсер продолжит с места остановки.

## Результаты:

Все данные в `data/steam_data.db`:
- `ccu_history` - история игроков
- `price_history` - история цен
- `app_status` - статусы обработки
- `errors` - ошибки для повторной обработки
